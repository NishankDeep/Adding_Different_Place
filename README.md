# **Adding_Different_Place**
The web app simply allow the user to add the the places they liked . The use can also look after the the places added
by the other users.

***

## TekStack Used
* Frontend TekStack
  + React Js
  
* Backend TekStack
  + Node JS
  + Express
  
* Database
  + MongoDB
